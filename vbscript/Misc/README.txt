Utilities for managing monitors:
- DumpExtMonRes.vbs : dumps External Monitors results to a CSV file
- DumpIntMonRes.vbs : dumps Internal Monitors agents results to a CSV file
- KeyManager.vbs : manages keys in configuration file
- ListInternals.vbs : lists Internal Monitors agents and their monitored values
- ResumeExt.vbs : resumes one or more suspended External Monitor
- SuspendExt.vbs : suspends one or more External Monitor
