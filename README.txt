This repository contains sample scripts to interact with Monitis API (http://monitis.com/api/api.html). 

Monitis (http://www.monitis.com) is a monitoring and notifications platform, which could be used to monitor almost anything. The scripts grouped per languages:


PowerShell
- Exchange2010 : create a custom monitor and then upload the current number of queued messages

VBScript

- BulkAdd : allows you to bulk add several External Monitors in a batch entering them in an Excel spreadsheet, 
            exporting to CSV and then process the exported file
- SQL : create a custom monitor and then upload values obtained from a SQL Server query
- WMI : create a custom monitor and then upload values obtained from a WMI query
